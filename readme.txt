Esta aplicacion se encuentra realizada para su uso exclusivamente academico con el objetivo del Trabajo Practico Integrador para la asignatura 
Programacion de aplicaciones visuales II

Unidad 1: Introducción a las aplicaciones Web y HTML5
Resultados de Aprendizaje: Comprender el funcionamiento de una aplicación web, la arquitectura de hardware
y software necesarios para la publicaciones de aplicaciones y servicios web, tecnologías utilizadas en entorno de
servidor y de equipo del cliente. Además se busca visualizar por parte del alumno el funcionamiento del
protocolo HTTP, cómo se compone un documento HTML, y que aprenda a elaborar páginas web y formularios
utilizando HTML5. 

Unidad 2: Diseño del frontend con CSS3 y Bootstrap
Resultados de Aprendizaje: Adquirir las competencias necesarias para poder construir y modificar la apariencia
de los documentos html que constituyen la interfaz gráfica de una aplicación web utilizando el estándar CSS3 y
Bootstrap

Unidad 3: Javascript y JQuery
Resultados de Aprendizaje: Adquirir el conocimiento y las habilidades necesarias para poder incorporar
comportamiento en el front end programando del lado del cliente con el lenguaje Javascript (estándar
ECMAScript) y el framework JQuery para la realización de páginas web dinámicas.

Unidad 4: AngularJS
Objetivo: Adquirir conocimientos y técnicas necesarias para el desarrollo de aplicaciones web con el patrón
MVC del lado del cliente utilizando el framework AngularJS.

Unidad 6: Acceso a datos con Entity Framework y LINQ
Resultados de Aprendizaje: Comprender las características y ventajas que ofrece Entity Framework para
agilizar el desarrollo del código de acceso a datos. Desarrollar las habilidades necesarias para utilizar EF y LINQ
para acceder y modificar datos almacenados en una base de datos.

Unidad 7: Implementando el backend con WebApi
Resultados de Aprendizaje: Adquirir las habilidades para desarrollar servicios web basados en el standard
WebApi orientado a integrar el front end de una aplicación web con el back end desarrollado en un entorno .NET.
